# Sell Exa Desktop App

A modern, high-performance desktop application for managing sales, orders, and administrative workflows. Built with React, TypeScript, and Electron, it offers a seamless experience with real-time updates and an intuitive interface.

## 🚀 Key Features

- **Order Management**: Real-time tracking and management of customer orders.
- **Product & Inventory**: Comprehensive tools for managing product listings and stock.
- **Admin Dashboard**: Specialized pages for managing administrators, companies, and sellers.
- **Audit Logs**: Detailed activity logs for security and tracking.
- **User Profiles**: Personalized profiles for users with secure authentication.
- **Auto-Updates**: Seamless background updates to ensure you're always on the latest version.
- **Interactive Maps**: Geographic visualization using Leaflet for orders and company locations.

## 🛠️ Technology Stack

- **Frontend**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Desktop Framework**: [Electron](https://www.electronjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Notifications**: [React Hot Toast](https://react-hot-toast.com/)
